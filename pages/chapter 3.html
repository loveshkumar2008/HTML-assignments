Chapter 3:lists

Lists

There are lots of occasions when we need to use lists. HTML provides us with three different types:

Ordered lists
Ordered lists are lists where each item in the list is numbered. For example, the list might be a set of steps for a recipe that must be performed in order, or a legal contract where each point needs to be identified by a section number.eg:
  1. Noun
  2. Pronoun
  3. Adjective
  4. Verb
  5. Adverb
  6. Preposition
  7. Conjunction
  8. interjection
Unordered lists
Unordered lists are lists that begin with a bullet point (rather than characters that indicate order)eg:
  • pen
  • pencil
  • copy
  • book
  • geometry box
  • sharpner
  • eraser
  • bag
  • water bottle
Definition lists
Definition lists are made up of a set of terms along with the definitions for each of those terms.definition list eg:
zoology
The branch of biology which deals with the study of animals.
botany
The branch of biology which deals with the study of plants.
microbiology
the branch biology which deals with the study of micro organisms.
Nested list
You can put a second list inside an li element to create a sublist or nested list. Browsers display nested lists indented further than the parent list. In nested unordered lists, the browser will usually change the style of the bullet point too.nested list eg:
  • biology
    • zoology
    • botany
    • microbiology
  • chemistry
    • organic chemistry
    • inorganic chemistry
    • physical chemistry
    • nuclear chemistry
    • bio chemistry
  • physics
    • nuclear physics
    • atomic physics
    • astro physics
